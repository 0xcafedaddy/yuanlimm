# 援力满满 - 二次元虚拟人物交易交易所

## 简介

https://www.yuanlimm.com/#/about

## 描述

本项目是二次元虚拟人物交易交易所的开源代码，供Rails爱好者学习，和玩家审查所用。

## 基础架构

Rails 5.2 + Mysql5.7 + Redis4.0

## 周边服务

搜索： 阿里云开放搜索

# 高速许愿工具

https://github.com/bydmm/yuanlimm-cli
